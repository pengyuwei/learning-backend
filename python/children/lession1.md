# 第一课：概念

这是一个家长教学孩子编程入门的教程。适合针对小学年龄阶段或者从来没接触过计算机的孩子的编程教学。

## 环境准备（家长）

Win10的Store里安装wsl。wsl里执行

```
python
```

## 课程内容

让孩纸知道什么是写代码
父母直接进入python立即模式，从1+1开始让孩子对着输入，然后出题让孩子计算


### 第一步

```
print 1+1
print 16*48
print '*' * 42
```

### 第二步


让猫等于狗，然后看看猫

让狗等于猫，然后看看狗

```
cat='dog'
print cat

print dog
```
